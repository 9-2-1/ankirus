这是一个可视化 Anki 卡片的工具。

不需要修改后端 python 部分——当时修改这部分的时候特地参考了Anki源代码，里面有很多坑，暂时先不要优化后端

只优化 Web 前端。

python 部分给出了一个单独的API：获取所有的卡片和状态 (ankirus/__main__.py:124)。

前端需要做到：

- 根据FSRS保留率，按照卡片类别分组（分组类似文件夹，是树形的，类似 数学::线性代数::矩阵），美观地进行复习完成度可视化
- 使用TreeMap可视化。（可以直观地看到哪一块做好了，哪一块有问题）
- 点击图块的时候能够预览卡片

一般想到的排布：

宽屏幕：左右拆分
[TreeMap | 详情]
窄屏幕：上下拆分
[TreeMap]
[详情]

详情部分需要展示选中的卡片所在的组名，保留率，问题，答案。答案默认隐藏，点击‘显示答案’按钮后显示。
